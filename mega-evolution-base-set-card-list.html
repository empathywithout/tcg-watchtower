Mega Evolution
Base Set

Complete guide to Mega Evolution Base Set — full card list, chase cards ranked by market price, and where to buy sealed product.

Mega Evolution Base Set is the first expansion of the Pokémon TCG Mega Evolution era, released September 26, 2025. Based on Japan's Mega Brave and Mega Symphonia sets, it introduced Mega Evolution Pokémon ex — one for each type — to the modern card game for the first time. The 188-card set features ten Mega Evolution Pokémon ex including Mega Lucario ex, Mega Gardevoir ex, and Mega Venusaur ex as the headline Megas. Mega Lucario ex MHR and Mega Gardevoir ex MHR are the top chase pulls in the set. This complete Mega Evolution card list includes every card with rarity filters and live prices updated daily.

A complete Mega Evolution master set contains 320 cards — 188 main set and secret rare cards plus 132 reverse holos (one for each Common, Uncommon, Rare, and Double Rare). Secret rares do not have reverse holo variants.

Mega Evolution FAQ

What is the most expensive Mega Evolution card?

The most expensive Mega Evolution Base Set cards are the Mega Lucario ex Mega Hyper Rare and Mega Gardevoir ex Mega Hyper Rare — both among the rarest pulls in the set. The Mega Lucario ex SIR and Mega Gardevoir ex SIR are also strong chase pulls. Check live prices on TCG Watchtower.

See all Mega Evolution chase cards ranked by price →

How many cards are in the Mega Evolution card list?

Mega Evolution Base Set contains 188 cards — 132 main set cards plus 56 secret rares including Illustration Rares, Ultra Rares, Special Illustration Rares, and two Mega Hyper Rares.

When did Mega Evolution release?

Mega Evolution Base Set released September 26, 2025 globally. The EMEA release was delayed to October 10, 2025 due to delivery issues.

What Mega Pokemon are in Mega Evolution Base Set?

Mega Evolution Base Set features ten Mega Evolution Pokemon ex — one for each type — including Mega Lucario ex, Mega Gardevoir ex, Mega Venusaur ex, Mega Latias ex, Mega Kangaskhan ex, and Mega Absol ex.

What Japanese sets is Mega Evolution based on?

Mega Evolution Base Set is based on Japan's Mega Brave and Mega Symphonia sets, which released August 1, 2025.
